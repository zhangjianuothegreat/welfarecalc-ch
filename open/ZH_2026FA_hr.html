U Švicarskoj su doplatci za djecu zakonski regulirani (FamZG). Naš kalkulator koristi specifične kantonalne odredbe Kantona Zürich za 2026. godinu kako bi precizno utvrdio vaše pravo. Kanton Zürich pritom primjenjuje posebnost s dobnom gradacijom od navršene 12. godine života.

1. Mjesečni iznosi (2026)

Doplatak za djecu (do 12 godina)

CHF 215 / mjesec

Za djecu od rođenja do navršene 12. godine života.

Doplatak za djecu (od 12 godina)

CHF 268 / mjesec

Specifični dobni stupanj Züricha: Od 12. do navršene 16. godine života.

Doplatak za obrazovanje

CHF 268 / mjesec

Od 16 godina do kraja obrazovanja (maksimalno do 25 godina).

2. Način izračuna

1

Određivanje kategorije i dobnog stupnja

Algoritam provjerava točnu dob i status obrazovanja svakog djeteta:

  • 0 do 12 godina: Pravo na redovni doplatak za djecu od CHF 215.
  • 12 do 16 godina: Pravo na povišeni dobni stupanj Züricha od CHF 268.
  • 16 do 25 godina: Pravo na doplatak za obrazovanje od CHF 268 (pod uvjetom dokazanog obrazovanja).
  • Djeca nesposobna za rad: Doplatak za djecu u iznosu od CHF 268 do navršene 20. godine života u slučaju trajne nesposobnosti za rad.
2

Izračun mjesečnog ukupnog iznosa

Zbroj se linearno formira iz odgovarajućih iznosa, budući da u Kantonu Zürich ne postoji povećanje za višečlane obitelji (od 3. djeteta):

(Broj djece ≤ 12 god. × CHF 215) + (Broj djece > 12 god. × CHF 268) + (Broj na obrazovanju × CHF 268)
3

Jednokratni doplatci (rođenje/adopcija)

Kanton Zürich, za razliku od nekih zapadnošvicarskih kantona, ne odobrava jednokratne doplatke za rođenje ili adopciju.

Vrsta doplatka Iznos (ZH)
Doplatak za rođenje CHF 0
Doplatak za adopciju CHF 0
4

Utvrđivanje godišnjeg prava

Isplata se obično vrši mjesečno putem poslodavca ili izravno putem nadležnog fonda za izravnanje obiteljskih doplataka:

Godišnji iznos = (Mjesečno ukupno pravo) × 12

3. Tehnički isječak koda (welfarecalc.ch Core-Engine)

Ovdje je službena JavaScript implementacija logike izračuna koja se izvršava na našim poslužiteljima i lokalno u vašem pregledniku:

// Kanton Zürich (ZH) Algoritam doplataka za djecu 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Do 12 godina const childOver12PerMonth = 268; // 12 do 16 godina const baseEduPerMonth = 268; // Od 16 godina na obrazovanju if (numChildren > 0) { // Defenzivna validacija dobne skupine > 12 godina const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Nema doplataka za rođenje/adopciju u ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
