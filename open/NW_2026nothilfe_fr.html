1. Paramètres de base de l'aide d'urgence (2026)

Taux journalier de base (personne seule)

CHF 8.00 / jour

Hébergement collectif, repas et besoins de base inclus.

Facteur de dégression pour couple

1.68 (pour la deuxième personne)

Pour deux adultes dans le même établissement (dégressif).

Supplément pour enfant (montant de base)

52% du taux de base ≈ CHF 4.16 / jour par enfant

Jusqu'à 18 ans, indépendamment du nombre d'enfants.

+10% du taux de base (CHF 0.80) par enfant supplémentaire

Montant forfaitaire supplémentaire à partir du troisième enfant.

2. Méthode de calcul (étape par étape)

1

Détermination du taux journalier pour adultes (dégression)

Le montant de base quotidien est d'abord établi en fonction du nombre d'adultes. Le canton de Nidwald applique un modèle dégressif pour refléter les économies d'échelle dans le ménage :

Personne seule : CHF 8.00 par jour
Couple (2 adultes) : CHF 8.00 + 8.00 × 1.68 = CHF 13.44 par jour
Pour plus de 2 adultes : chaque personne supplémentaire reçoit CHF 8.00 × 1.68 (augmentation dégressive).

Pratique conforme aux directives du canton de Nidwald : les effets de synergie sont pris en compte équitablement.

2

Suppléments pour enfants et bonus familles nombreuses

Pour chaque enfant (< 18 ans), un supplément de 52% du taux de base adulte (CHF 8.00) est ajouté. Afin de soutenir les familles avec plusieurs enfants, le canton de Nidwald accorde un bonus supplémentaire de 10% du taux de base par enfant supplémentaire à partir du troisième enfant.

Montant de base enfants = Nombre d'enfants × (CHF 8.00 × 0.52)
Bonus dès le 3e enfant : (Nombre d'enfants - 2) × (CHF 8.00 × 0.10)
Montant total journalier = Taux adulte + Montant de base enfants + Bonus
3

Projection mensuelle et arrondi

L'aide d'urgence est calculée sur la base de 30.42 jours (mois moyen). Le montant final est arrondi commercialement à 5 centimes (0.05 CHF), conformément à la pratique cantonale.

Droit mensuel = ARRONDI( (Montant total journalier × 30.42) ; 0.05 )
Règle d'arrondiExemple

Informations complémentaires importantes sur l'aide d'urgence dans le canton de Nidwald :
• L'aide d'urgence est principalement fournie sous forme de prestations en nature (logement, repas) ; la part en espèces est réduite au minimum.
• Les avoirs dépassant le montant exonéré (CHF 5'000 par personne / max. CHF 15'000 par ménage) doivent être utilisés en priorité.
• Les personnes sans autorisation de séjour avec une décision de renvoi exécutoire n'ont droit qu'à l'aide d'urgence, et non à l'aide sociale ordinaire.
• Le bonus à partir du 3e enfant suit la politique sociale de soutien aux familles nombreuses dans le canton de Nidwald.
