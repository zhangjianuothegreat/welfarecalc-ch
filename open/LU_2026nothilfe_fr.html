La transparence crée la confiance. Dans le canton de Lucerne (LU), l'aide d'urgence pour les personnes en situation de détresse est calculée selon un modèle mathématique précis. Nous divulguons ici les paramètres officiels et l'algorithme de calcul pour l'année 2026.

1. Paramètres de calcul (Lucerne 2026)

Taux journalier de base (taux individuel)

CHF 12.80 / jour

Besoin de base pour une personne seule (l'un des taux les plus élevés de Suisse).

Facteur de couple (dégressif)

Facteur 1.75

Deux adultes dans le ménage reçoivent 175% du taux de base.

Supplément pour enfants

Facteur 0.58 / 0.15

58% du taux de base par enfant + 15% de bonus supplémentaire à partir du 3e enfant.

2. Méthode de calcul (étape par étape)

1

Détermination du taux journalier pour adultes

Le taux journalier pour les adultes est basé sur le nombre de personnes dans le ménage bénéficiaire :

Personne seule : CHF 12.80 / jour
Couple (2 adultes) : CHF 12.80 + (1 × 12.80 × 1.75) = CHF 35.20 / jour

Remarque : Pour plus de 2 adultes, chaque personne supplémentaire est ajoutée selon la clé cantonale avec le taux dégressif.

2

Supplément pour enfants et bonus pour grandes familles

Lucerne accorde un taux de base pour les enfants ainsi qu'un bonus supplémentaire à partir du 3e enfant :

Taux de base par enfant : CHF 12.80 × 0.58 = CHF 7.42 / jour
Bonus (dès le 3e enfant) : CHF 12.80 × 0.15 = CHF 1.92 / jour supplémentaire
3

Projection mensuelle et arrondi

Les droits journaliers sont projetés sur le mois statistique (30,42 jours) et arrondis commercialement.

Montant mensuel = ARRONDI( (Montant total journalier × 30,42) ; 0 )
Particularités à Lucerne :
• Forme de logement : Le taux de CHF 12.80 s'applique principalement aux concepts de "mixed housing".
• Formation : Les enfants en formation sont pris en compte au taux enfant jusqu'à l'obtention du diplôme (max. 25 ans).
• Subsidiarité : Les propres biens ou revenus sont à prendre en compte prioritairement.
