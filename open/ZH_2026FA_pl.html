W Szwajcarii zasiłki rodzinne są regulowane ustawowo (FamZG). Nasz kalkulator wykorzystuje specyficzne przepisy kantonalne kantonu Zurych na rok 2026, aby precyzyjnie określić Twoje uprawnienia. Kanton Zurych stosuje przy tym szczególne rozwiązanie z progresją wiekową od ukończonego 12. roku życia.

1. Stawki miesięczne (2026)

Zasiłek na dziecko (do 12 lat)

CHF 215 / miesiąc

Dla dzieci od urodzenia do ukończenia 12. roku życia.

Zasiłek na dziecko (od 12 lat)

CHF 268 / miesiąc

Specyficzna zuryska progresja wiekowa: Od 12. do ukończenia 16. roku życia.

Zasiłek edukacyjny

CHF 268 / miesiąc

Od 16 lat do końca kształcenia (maksymalnie do 25 lat).

2. Sposób obliczania

1

Określenie kategorii i progu wiekowego

Algorytm sprawdza dokładny wiek i status edukacyjny każdego dziecka:

  • 0 do 12 lat: Prawo do regularnego zasiłku na dziecko w wysokości CHF 215.
  • 12 do 16 lat: Prawo do podwyższonej zuryskiej stawki wiekowej w wysokości CHF 268.
  • 16 do 25 lat: Prawo do zasiłku edukacyjnego w wysokości CHF 268 (pod warunkiem udokumentowanego kształcenia).
  • Dzieci niezdolne do pracy: Zasiłek na dziecko w wysokości CHF 268 do ukończenia 20. roku życia w przypadku trwałej niezdolności do pracy.
2

Obliczenie miesięcznej kwoty całkowitej

Suma tworzona jest liniowo z odpowiednich stawek, ponieważ w kantonie Zurych nie istnieje podwyższenie dla rodzin wielodzietnych (od 3. dziecka):

(Liczba dzieci ≤ 12 lat × CHF 215) + (Liczba dzieci > 12 lat × CHF 268) + (Liczba w kształceniu × CHF 268)
3

Zasiłki jednorazowe (urodzenie/adopcja)

Kanton Zurych, w przeciwieństwie do niektórych kantonów zachodnioszwajcarskich, nie przyznaje jednorazowych zasiłków z tytułu urodzenia lub adopcji.

Rodzaj zasiłku Kwota (ZH)
Zasiłek z tytułu urodzenia CHF 0
Zasiłek z tytułu adopcji CHF 0
4

Ustalenie rocznego roszczenia

Wypłata następuje zazwyczaj miesięcznie za pośrednictwem pracodawcy lub bezpośrednio przez właściwy kantonalny fundusz wyrównawczy:

Kwota roczna = (Miesięczne roszczenie całkowite) × 12

3. Techniczny fragment kodu (silnik główny welfarecalc.ch)

Oto oficjalna implementacja JavaScript logiki obliczeniowej, która jest wykonywana na naszych serwerach i lokalnie w Twojej przeglądarce:

// Kanton Zurych (ZH) Algorytm zasiłków rodzinnych 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Do 12 lat const childOver12PerMonth = 268; // 12 do 16 lat const baseEduPerMonth = 268; // Od 16 lat w kształceniu if (numChildren > 0) { // Defensywna walidacja grupy wiekowej > 12 lat const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Brak zasiłków z tytułu urodzenia/adopcji w ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Uprawnienia i priorytet:
Tylko jeden rodzic może pobierać zasiłek rodzinny na to samo dziecko. Jeśli oboje rodzice pracują zawodowo, ustawowa kolejność priorytetów zgodnie z art. 12 FamZG ściśle reguluje pierwszeństwo roszczenia (według miejsca pracy, pieczy nad dzieckiem lub wyższego dochodu).
