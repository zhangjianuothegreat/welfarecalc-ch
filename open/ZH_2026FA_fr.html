En Suisse, les allocations familiales sont régies par la loi (LAFam). Notre calculateur utilise les dispositions cantonales spécifiques du canton de Zurich pour 2026 afin de déterminer précisément votre droit. Le canton de Zurich applique ici une particularité avec un palier d'âge à partir de 12 ans révolus.

1. Montants mensuels (2026)

Allocation pour enfant (jusqu'à 12 ans)

CHF 215 / mois

Pour les enfants de la naissance jusqu'à 12 ans révolus.

Allocation pour enfant (dès 12 ans)

CHF 268 / mois

Palier d'âge spécifique zurichois : De 12 à 16 ans révolus.

Allocation de formation

CHF 268 / mois

Dès 16 ans jusqu'à la fin de la formation (maximum 25 ans).

2. Le mode de calcul

1

Détermination de la catégorie et du palier d'âge

L'algorithme vérifie l'âge exact et le statut de formation de chaque enfant :

  • 0 à 12 ans : Droit à l'allocation pour enfant régulière de CHF 215.
  • 12 à 16 ans : Droit au palier d'âge zurichois majoré de CHF 268.
  • 16 à 25 ans : Droit à l'allocation de formation de CHF 268 (si formation prouvée).
  • Enfants incapables de travailler : Allocation pour enfant d'un montant de CHF 268 jusqu'à 20 ans révolus en cas d'incapacité de travail permanente.
2

Calcul du montant mensuel total

La somme se forme linéairement à partir des taux respectifs, car il n'existe pas de majoration pour les familles nombreuses (dès le 3ème enfant) dans le canton de Zurich :

(Nombre d'enfants ≤ 12 ans × CHF 215) + (Nombre d'enfants > 12 ans × CHF 268) + (Nombre en formation × CHF 268)
3

Allocations uniques (naissance/adoption)

Contrairement à certains cantons romands, le canton de Zurich n'accorde pas d'allocations uniques de naissance ou d'adoption.

Type d'allocation Montant (ZH)
Allocation de naissance CHF 0
Allocation d'adoption CHF 0
4

Détermination du droit annuel

Le versement a généralement lieu mensuellement par l'employeur ou directement par la caisse de compensation familiale compétente :

Montant annuel = (Droit mensuel total) × 12

3. Extrait de code technique (Moteur central welfarecalc.ch)

Voici l'implémentation JavaScript officielle de la logique de calcul, exécutée sur nos serveurs et localement dans votre navigateur :

// Canton de Zurich (ZH) Algorithme des allocations familiales 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Jusqu'à 12 ans const childOver12PerMonth = 268; // 12 à 16 ans const baseEduPerMonth = 268; // Dès 16 ans en formation if (numChildren > 0) { // Validation défensive du groupe d'âge > 12 ans const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Pas d'allocations de naissance/adoption dans ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Droit et priorité :
Un seul parent peut percevoir l'allocation familiale pour le même enfant. Si les deux parents exercent une activité lucrative, l'ordre de priorité légal selon l'art. 12 LAFam régit strictement le droit primaire (selon le lieu de travail, la garde ou le revenu le plus élevé).
