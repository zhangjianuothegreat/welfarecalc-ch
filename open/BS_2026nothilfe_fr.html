Bâle-Ville mise sur un modèle transparent pour garantir le minimum vital. Nous dévoilons ici l'algorithme officiel pour 2026. Par rapport à d'autres cantons, Bâle-Ville se distingue par un montant journalier de base plus élevé, afin de tenir compte du niveau de vie urbain.

1. Paramètres centraux (BS 2026)

Montant journalier de base (personne seule)

CHF 11.00 / jour

Montant le plus élevé de Suisse, y compris l'argent de poche pour les besoins personnels.

Facteur de couple (dégressif)

Facteur 1.80

Les ménages communs reçoivent 180% du montant de base (CHF 19.80 pour 2 pers.).

Facteur supplément enfant

Facteur 0.60

Montant supplémentaire par enfant : 60% du montant de base (CHF 6.60 / jour).

2. Démarche de calcul

1

Détermination des besoins adultes

Bâle-Ville utilise un modèle dégressif pour les ménages de plusieurs personnes :

Personne seule : CHF 11.00 / jour
Couple/Plusieurs personnes : 11.00 + (adultes - 1) × 11.00 × 0.80

Remarque : À partir de la deuxième personne adulte, le besoin est pondéré à 80%.

2

Ajouter la prestation forfaitaire pour enfants

Chaque enfant reçoit un facteur fixe de 0.60 sur le montant de base, indépendamment du nombre :

Total enfants = Nombre d'enfants × (11.00 × 0.60) = Nombre × CHF 6.60 / jour
3

Calcul mensuel et arrondi

Le montant journalier est multiplié par le facteur standard suisse de 30.42. Le montant final est arrondi au franc supérieur ou inférieur selon la règle commerciale.

Droit mensuel = ARRONDI( (Total journalier × 30.42) ; 0 )
Remarques importantes pour Bâle-Ville :
• Les montants s'appliquent en cas d'hébergement dans des centres d'accueil cantonaux.
• Les primes d'assurance maladie et les soins médicaux d'urgence sont pris en charge séparément.
• En cas d'absence injustifiée, le montant en espèces peut être réduit.
