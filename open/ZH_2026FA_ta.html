FAMZG-இணக்கமான 2026

FA-அல்காரிதம்: சூரிச் கேன்டன்

குடும்ப கொடுப்பனவுகளின் (குழந்தை மற்றும் பயிற்சி கொடுப்பனவுகள்) வெளிப்படையான ஆவணப்படுத்தல்

⚡

விரைவான சரிபார்ப்பு: உங்களுக்கு உரிமை உள்ளதா?

காப்பீட்டுக் கட்டணக் குறைப்பு (IPV), கூடுதல் நலன்கள் அல்லது பிற சமூக உதவிகளுக்கு உங்களுக்கு உரிமை உள்ளதா என விரைவாகச் சரிபார்க்கவும்.

இலவசமாக சரிபார்க்கவும் →

சுவிட்சர்லாந்தில், குடும்ப கொடுப்பனவுகள் சட்டப்பூர்வமாக (FamZG) கட்டுப்படுத்தப்படுகின்றன. எங்கள் கால்குலேட்டர், உங்கள் உரிமையை துல்லியமாக தீர்மானிக்க, 2026 ஆம் ஆண்டிற்கான சூரிச் கேன்டனின் குறிப்பிட்ட கேன்டனல் விதிமுறைகளைப் பயன்படுத்துகிறது. சூரிச் கேன்டன், 12 வயது நிறைவடைந்ததிலிருந்து வயது படிநிலையுடன் ஒரு சிறப்பு அம்சத்தைப் பயன்படுத்துகிறது.

1. மாதாந்திர விகிதங்கள் (2026)

குழந்தை கொடுப்பனவு (12 வயது வரை)

CHF 215 / மாதம்

பிறப்பு முதல் 12 வயது நிறைவடையும் வரையிலான குழந்தைகளுக்கு.

குழந்தை கொடுப்பனவு (12 வயது முதல்)

CHF 268 / மாதம்

குறிப்பிட்ட சூரிச் வயது நிலை: 12 வயது முதல் 16 வயது நிறைவடையும் வரை.

பயிற்சி கொடுப்பனவு

CHF 268 / மாதம்

16 வயது முதல் பயிற்சி முடியும் வரை (அதிகபட்சம் 25 வயது வரை).

2. கணக்கீட்டு முறை

1

வகை மற்றும் வயது நிலையை தீர்மானித்தல்

அல்காரிதம் ஒவ்வொரு குழந்தையின் சரியான வயது மற்றும் பயிற்சி நிலையை சரிபார்க்கிறது:

  • 0 முதல் 12 வயது வரை: CHF 215 என்ற வழக்கமான குழந்தை கொடுப்பனவுக்கான உரிமை.
  • 12 முதல் 16 வயது வரை: CHF 268 என்ற உயர்த்தப்பட்ட சூரிச் வயது நிலைக்கான உரிமை.
  • 16 முதல் 25 வயது வரை: CHF 268 பயிற்சி கொடுப்பனவுக்கான உரிமை (நிரூபிக்கப்பட்ட பயிற்சியில் இருந்தால்).
  • வேலை செய்ய இயலாத குழந்தைகள்: நிரந்தர வேலை செய்ய இயலாமை ஏற்பட்டால், 20 வயது நிறைவடையும் வரை CHF 268 தொகையில் குழந்தை கொடுப்பனவு.
2

மாதாந்திர மொத்த தொகையின் கணக்கீடு

சூரிச் கேன்டனில் அதிக குழந்தைகள் உள்ள குடும்பங்களுக்கு (3வது குழந்தையிலிருந்து) எந்த உயர்வும் இல்லாததால், மொத்த தொகை ஒவ்வொரு விகிதத்திலிருந்தும் நேர்கோட்டில் உருவாகிறது:

(12 வயதுக்குட்பட்ட குழந்தைகளின் எண்ணிக்கை × CHF 215) + (12 வயதுக்கு மேற்பட்ட குழந்தைகளின் எண்ணிக்கை × CHF 268) + (பயிற்சியில் உள்ளவர்களின் எண்ணிக்கை × CHF 268)
3

ஒருமுறை கொடுப்பனவுகள் (பிறப்பு/தத்தெடுப்பு)

சில மேற்கு சுவிஸ் கேன்டன்களைப் போலல்லாமல், சூரிச் கேன்டன் எந்த ஒருமுறை பிறப்பு அல்லது தத்தெடுப்பு கொடுப்பனவுகளையும் வழங்காது.

கொடுப்பனவு வகை தொகை (ZH)
பிறப்பு கொடுப்பனவு CHF 0
தத்தெடுப்பு கொடுப்பனவு CHF 0
4

வருடாந்திர உரிமையை தீர்மானித்தல்

பணம் செலுத்துதல் பொதுவாக மாதந்தோறும் முதலாளி மூலமாகவோ அல்லது பொறுப்பான குடும்ப இழப்பீட்டு நிதி மூலமாகவோ நடைபெறுகிறது:

வருடாந்திர தொகை = (மாதாந்திர மொத்த உரிமை) × 12

3. தொழில்நுட்ப குறியீடு பகுதி (welfarecalc.ch Core-Engine)

எங்கள் சேவையகங்களிலும் உங்கள் உலாவியிலும் உள்நாட்டிலும் இயக்கப்படும் கணக்கீட்டு தர்க்கத்தின் அதிகாரப்பூர்வ JavaScript செயலாக்கம் இங்கே உள்ளது:

// சூரிச் கேன்டன் (ZH) குடும்ப கொடுப்பனவுகள் அல்காரிதம் 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // 12 வயது வரை const childOver12PerMonth = 268; // 12 முதல் 16 வயது வரை const baseEduPerMonth = 268; // 16 வயது முதல் பயிற்சியில் if (numChildren > 0) { // 12 வயதுக்கு மேற்பட்ட வயதினரின் தற்காப்பு சரிபார்ப்பு const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // ZH இல் பிறப்பு/தத்தெடுப்பு கொடுப்பனவுகள் இல்லை return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
உரிமை மற்றும் முன்னுரிமை:
ஒரே குழந்தைக்கு ஒரு பெற்றோர் மட்டுமே குடும்ப கொடுப்பனவைப் பெற முடியும். இரண்டு பெற்றோரும் வேலை செய்தால், FamZG இன் கட்டுரை 12 இன் படி சட்டப்பூர்வ முன்னுரிமை வரிசை முதல் உரிமையை (வேலை இடம், பாதுகாப்பு அல்லது அதிக வருமானத்தின் அடிப்படையில்) கண்டிப்பாக ஒழுங்குபடுத்துகிறது.
கால்குலேட்டரைத் தொடங்கவும் அதிகாரப்பூர்வ SVA சூரிச் பக்கம்
