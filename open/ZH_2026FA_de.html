In der Schweiz sind Familienzulagen gesetzlich geregelt (FamZG). Unser Rechner nutzt die spezifischen kantonalen Bestimmungen des Kantons Zürich für 2026, um Ihren Anspruch präzise zu ermitteln. Der Kanton Zürich wendet hierbei eine Besonderheit mit einer Altersabstufung ab dem vollendeten 12. Altersjahr an.

1. Monatssätze (2026)

Kinderzulage (bis 12 Jahre)

CHF 215 / Monat

Für Kinder ab Geburt bis zum vollendeten 12. Altersjahr.

Kinderzulage (ab 12 Jahre)

CHF 268 / Monat

Spezifische Zürcher Altersstufe: Vom 12. bis zum vollendeten 16. Altersjahr.

Ausbildungszulage

CHF 268 / Monat

Ab 16 Jahren bis zum Ende der Ausbildung (maximal bis 25 Jahre).

2. Der Berechnungsweg

1

Bestimmung der Kategorie und Altersstufe

Der Algorithmus prüft das genaue Alter und den Ausbildungsstatus jedes Kindes:

  • 0 bis 12 Jahre: Anspruch auf die reguläre Kinderzulage von CHF 215.
  • 12 bis 16 Jahre: Anspruch auf die erhöhte Zürcher Altersstufe von CHF 268.
  • 16 bis 25 Jahre: Anspruch auf die Ausbildungszulage von CHF 268 (sofern in nachgewiesener Ausbildung).
  • Erwerbsunfähige Kinder: Kinderzulage in Höhe von CHF 268 bis zum vollendeten 20. Altersjahr bei dauerhafter Erwerbsunfähigkeit.
2

Berechnung des monatlichen Gesamtbetrags

Die Summe bildet sich linear aus den jeweiligen Sätzen, da im Kanton Zürich keine Erhöhung für kinderreiche Familien (ab dem 3. Kind) existiert:

(Anzahl Kinder ≤ 12 J. × CHF 215) + (Anzahl Kinder > 12 J. × CHF 268) + (Anzahl in Ausbildung × CHF 268)
3

Einmalige Zulagen (Geburt/Adoption)

Der Kanton Zürich gewährt im Gegensatz zu einigen Westschweizer Kantonen keine einmaligen Geburts- oder Adoptionszulagen.

ZulagenartBetrag (ZH)
GeburtszulageCHF 0
AdoptionszulageCHF 0
4

Ermittlung des Jahresanspruchs

Die Auszahlung erfolgt in der Regel monatlich über den Arbeitgeber oder direkt über die zuständige Familienausgleichskasse:

Jahresbetrag = (Monatlicher Gesamtanspruch) × 12

3. Technischer Code-Auszug (welfarecalc.ch Core-Engine)

Hier ist die offizielle JavaScript-Implementierung der Berechnungslogik, die auf unseren Servern und lokal in Ihrem Browser ausgeführt wird:

// Kanton Zürich (ZH) Familienzulagen-Algorithmus 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Bis 12 Jahre const childOver12PerMonth = 268; // 12 bis 16 Jahre const baseEduPerMonth = 268; // Ab 16 Jahre in Ausbildung if (numChildren > 0) { // Defensive Validierung der Altersgruppe > 12 Jahre const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Keine Geburts-/Adoptionszulagen in ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Anspruchsberechtigung und Priorität:
Nur ein Elternteil kann die Familienzulage für dasselbe Kind beziehen. Wenn beide Elternteile erwerbstätig sind, regelt die gesetzliche Prioritätsordnung nach Art. 12 FamZG strikt den Erstanspruch (nach Erwerbsort, Obhut oder höherem Einkommen).
