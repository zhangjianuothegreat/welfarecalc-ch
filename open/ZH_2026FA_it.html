In Svizzera gli assegni familiari sono regolati per legge (LAFam). Il nostro calcolatore utilizza le specifiche disposizioni cantonali del Cantone Zurigo per il 2026 per determinare con precisione il suo diritto. Il Cantone Zurigo applica una particolarità con una fascia d'età a partire dal compimento del 12° anno di età.

1. Importi mensili (2026)

Assegno per figli (fino a 12 anni)

CHF 215 / mese

Per bambini dalla nascita fino al compimento del 12° anno di età.

Assegno per figli (da 12 anni)

CHF 268 / mese

Fascia d'età specifica zurighese: Dal 12° al compimento del 16° anno di età.

Assegno di formazione

CHF 268 / mese

Da 16 anni fino al termine della formazione (massimo fino a 25 anni).

2. Il procedimento di calcolo

1

Determinazione della categoria e della fascia d'età

L'algoritmo verifica l'età esatta e lo stato di formazione di ogni figlio:

  • Da 0 a 12 anni: Diritto all'assegno per figli regolare di CHF 215.
  • Da 12 a 16 anni: Diritto alla fascia d'età zurighese maggiorata di CHF 268.
  • Da 16 a 25 anni: Diritto all'assegno di formazione di CHF 268 (se in formazione comprovata).
  • Figli inabili al lavoro: Assegno per figli dell'importo di CHF 268 fino al compimento del 20° anno di età in caso di inabilità permanente al lavoro.
2

Calcolo dell'importo mensile totale

La somma si forma linearmente dalle rispettive aliquote, poiché nel Cantone Zurigo non esiste un aumento per famiglie numerose (dal 3° figlio):

(Numero figli ≤ 12 anni × CHF 215) + (Numero figli > 12 anni × CHF 268) + (Numero in formazione × CHF 268)
3

Assegni unici (nascita/adozione)

Il Cantone Zurigo, a differenza di alcuni cantoni della Svizzera romanda, non concede assegni unici di nascita o di adozione.

Tipo di assegno Importo (ZH)
Assegno di nascita CHF 0
Assegno di adozione CHF 0
4

Determinazione del diritto annuale

Il pagamento avviene di solito mensilmente tramite il datore di lavoro o direttamente tramite la cassa di compensazione familiare competente:

Importo annuale = (Diritto mensile totale) × 12

3. Estratto del codice tecnico (Motore centrale welfarecalc.ch)

Ecco l'implementazione JavaScript ufficiale della logica di calcolo, eseguita sui nostri server e localmente nel suo browser:

// Cantone Zurigo (ZH) Algoritmo assegni familiari 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Fino a 12 anni const childOver12PerMonth = 268; // Da 12 a 16 anni const baseEduPerMonth = 268; // Da 16 anni in formazione if (numChildren > 0) { // Validazione difensiva della fascia d'età > 12 anni const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Nessun assegno di nascita/adozione in ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Diritto e priorità:
Solo un genitore può ricevere l'assegno familiare per lo stesso figlio. Se entrambi i genitori lavorano, l'ordine di priorità legale secondo l'art. 12 LAFam regola rigorosamente il diritto primario (in base al luogo di lavoro, all'affidamento o al reddito più elevato).
