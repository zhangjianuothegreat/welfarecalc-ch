La confiance par la transparence. L'aide d'urgence (aide en cas de détresse) pour les personnes devant quitter le pays ou les personnes faisant l'objet d'une décision de renvoi exécutoire est calculée dans le canton de Soleure selon des critères légaux stricts. Nous dévoilons ici le modèle mathématique complet et l'algorithme pour l'année 2026.

1. Paramètres fondamentaux de l'aide d'urgence (SO 2026)

Taux journalier de base (Adulte)

CHF 8.00 / Jour

Argent de poche/argent liquide pour une personne seule dans un hébergement collectif.

Facteur de couple (Dégressivité)

Facteur 1.68

Deux personnes adultes dans une unité d'assistance reçoivent ensemble 168% du taux individuel.

Facteur de supplément pour enfant

Facteur 0.52

Montant supplémentaire par enfant (52% du taux de base adulte), indépendamment du nombre d'enfants.

2. Le mode de calcul (Étape par étape)

1

Détermination du taux journalier adulte

Le canton de Soleure utilise un modèle de dégressivité pour les ménages avec plusieurs adultes :

Si Adultes = 1 : CHF 8.00 / Jour
Si Adultes > 1 : CHF 8.00 + (Nombre d'adultes - 1) × 8.00 × 1.68

Remarque : Pour un couple/un partenariat, la deuxième personne adulte reçoit un taux dégressif de CHF 13.44 par jour.

2

Ajout du supplément pour enfant

Pour chaque enfant vivant dans le même ménage, un montant forfaitaire est ajouté, correspondant à 52% du taux de base adulte (soit CHF 4.16 par enfant/jour) :

Montant journalier pour enfant = Nombre d'enfants × (8.00 × 0.52) = Nombre d'enfants × CHF 4.16 / Jour

Le canton de Soleure n'accorde aucune progression supplémentaire ni bonus pour famille nombreuse à partir du 3ème enfant dans le cadre de l'aide d'urgence (le facteur reste fixe à 0.52).

3

Projection mensuelle & Arrondi

Le montant journalier total est projeté sur le facteur mensuel suisse standardisé de 30.42 jours. Le résultat est arrondi commercialement au franc supérieur (1.00 CHF), conformément à la pratique cantonale d'exécution pour le versement des avoirs d'aide d'urgence.

Droit mensuel = ARRONDI( (Montant journalier total × 30.42) ; 0 )
Particularités administratives importantes dans le canton de SO :
• Priorité aux prestations en nature : L'aide d'urgence est principalement fournie sous forme de prestations en nature (hébergement, soins médicaux de base, alimentation directe). Le montant calculé représente le minimum d'argent de poche/liquide.
• Exécution du renvoi : Les personnes faisant l'objet d'une décision de renvoi exécutoire n'ont plus droit, par la loi, à l'aide sociale ordinaire (SKOS), mais relèvent exclusivement de ce système minimal d'aide en cas de détresse.
• Subsidiarité : Les ressources financières propres (fortune/revenu) doivent être entièrement épuisées avant de pouvoir prétendre à l'aide d'urgence cantonale.
