Na Suíça, os abonos familiares são regulados por lei (FamZG). A nossa calculadora utiliza as disposições cantonais específicas do Cantão de Zurique para 2026, a fim de determinar com precisão o seu direito. O Cantão de Zurique aplica aqui uma particularidade com um escalão etário a partir dos 12 anos completos.

1. Valores Mensais (2026)

Abono por Filhos (até 12 anos)

CHF 215 / mês

Para crianças desde o nascimento até aos 12 anos completos.

Abono por Filhos (a partir de 12 anos)

CHF 268 / mês

Escalão etário específico de Zurique: Dos 12 aos 16 anos completos.

Abono de Formação

CHF 268 / mês

A partir dos 16 anos até ao final da formação (máximo até 25 anos).

2. O Método de Cálculo

1

Determinação da Categoria e Escalão Etário

O algoritmo verifica a idade exata e o estado de formação de cada criança:

  • 0 a 12 anos: Direito ao abono por filhos regular de CHF 215.
  • 12 a 16 anos: Direito ao escalão etário superior de Zurique de CHF 268.
  • 16 a 25 anos: Direito ao abono de formação de CHF 268 (desde que em formação comprovada).
  • Filhos incapacitados para o trabalho: Abono por filhos no valor de CHF 268 até aos 20 anos completos em caso de incapacidade laboral permanente.
2

Cálculo do Valor Total Mensal

A soma é formada linearmente a partir das respetivas taxas, uma vez que no Cantão de Zurique não existe nenhum aumento para famílias numerosas (a partir do 3.º filho):

(N.º filhos ≤ 12 a. × CHF 215) + (N.º filhos > 12 a. × CHF 268) + (N.º em formação × CHF 268)
3

Abonos Únicos (Nascimento/Adoção)

Ao contrário de alguns cantões da Suíça ocidental, o Cantão de Zurique não concede abonos únicos de nascimento ou adoção.

Tipo de Abono Valor (ZH)
Abono de Nascimento CHF 0
Abono de Adoção CHF 0
4

Determinação do Direito Anual

O pagamento é geralmente efetuado mensalmente através da entidade patronal ou diretamente pela caixa de compensação familiar competente:

Valor Anual = (Direito Total Mensal) × 12

3. Excerto de Código Técnico (Core-Engine welfarecalc.ch)

Aqui está a implementação oficial em JavaScript da lógica de cálculo, que é executada nos nossos servidores e localmente no seu navegador:

// Cantão de Zurique (ZH) Algoritmo de Abonos Familiares 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Até 12 anos const childOver12PerMonth = 268; // 12 a 16 anos const baseEduPerMonth = 268; // A partir de 16 anos em formação if (numChildren > 0) { // Validação defensiva do grupo etário > 12 anos const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // Sem abonos de nascimento/adoção em ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Direito e Prioridade:
Apenas um dos progenitores pode receber o abono familiar para o mesmo filho. Se ambos os progenitores trabalharem, a ordem de prioridade legal de acordo com o Art. 12 FamZG regula estritamente o direito primário (de acordo com o local de trabalho, guarda ou rendimento mais elevado).
