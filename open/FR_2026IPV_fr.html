Transparence radicale : Découvrez comment notre calculateur détermine la Réduction des Primes d'assurance-maladie (RPC) 2026 pour le canton de Fribourg. L'algorithme utilise le modèle officiel de lissage à 60 paliers de l'ECASFR.

1. Paramètres clés (2026)

Seuils de revenu (départ)

CHF 37'000 (personne seule)

Base pour le calcul du niveau de droit.

Majoration pour la fortune

5.0 %

Part de la fortune nette ajoutée au revenu.

2. Démarche de calcul

1

Revenu déterminant (LNA)

Le LNA est calculé à partir de vos données fiscales comme suit :

[Revenu net] + [5% de la fortune] - [Déduction pour enfant : CHF 5'000/enfant]

Les pensions alimentaires effectivement versées sont également déduites.

2

Primes de référence (mensuelles)

Le canton de Fribourg utilise pour 2026 les valeurs forfaitaires suivantes :

CatégoriePrime mensuelle
Adultes (dès 26 ans)CHF 495 / mois
Jeunes adultes (19-25 ans)CHF 415 / mois
Enfants (0-18 ans)CHF 136 / mois
3

Modèle de lissage (60 paliers)

Fribourg applique un lissage linéaire. D'abord, on calcule l'écart en pourcentage par rapport au seuil de revenu individuel :

Écart % = [(Montant seuil - LNA) / Montant seuil] × 100

Sur la base de cet écart, le taux de subvention (1% à 65%) est interpolé linéairement sur 60 paliers.

4

Garantie minimale pour les descendants

Indépendamment du taux calculé, le canton garantit :

CatégorieTaux garanti
Enfantsmin. 80 %
Jeunes en formationmin. 50 %
Critères d'exclusion :
Si la fortune nette dépasse CHF 250'000 ou si le revenu net dépasse CHF 150'000, il n'y a généralement pas droit à la RPC dans le canton de Fribourg.
