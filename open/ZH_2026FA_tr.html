İsviçre'de aile ödenekleri yasal olarak düzenlenmiştir (FamZG). Hesaplayıcımız, hakkınızı hassas bir şekilde belirlemek için Zürih Kantonu'nun 2026 yılına özgü kanton düzenlemelerini kullanır. Zürih Kantonu burada, 12. yaşın tamamlanmasından itibaren bir yaş kademelendirmesi özelliği uygular.

1. Aylık Oranlar (2026)

Çocuk Ödeneği (12 yaşına kadar)

CHF 215 / Ay

Doğumdan 12. yaşın tamamlanmasına kadar olan çocuklar için.

Çocuk Ödeneği (12 yaş ve üzeri)

CHF 268 / Ay

Zürih'e özgü yaş kademesi: 12. yaştan 16. yaşın tamamlanmasına kadar.

Eğitim Ödeneği

CHF 268 / Ay

16 yaşından itibaren eğitimin sonuna kadar (en fazla 25 yaşına kadar).

2. Hesaplama Yöntemi

1

Kategori ve Yaş Kademesinin Belirlenmesi

Algoritma, her çocuğun kesin yaşını ve eğitim durumunu kontrol eder:

  • 0 ila 12 yaş: CHF 215 tutarındaki normal çocuk ödeneği hakkı.
  • 12 ila 16 yaş: CHF 268 tutarındaki yüksek Zürih yaş kademesi hakkı.
  • 16 ila 25 yaş: CHF 268 tutarındaki eğitim ödeneği hakkı (kanıtlanmış bir eğitimde olması koşuluyla).
  • Çalışamaz durumdaki çocuklar: Kalıcı çalışamazlık durumunda 20. yaşın tamamlanmasına kadar CHF 268 tutarında çocuk ödeneği.
2

Aylık Toplam Tutarın Hesaplanması

Zürih Kantonu'nda geniş aileler için (3. çocuktan itibaren) bir artış olmadığından, toplam ilgili oranlardan doğrusal olarak oluşur:

(≤ 12 yaş çocuk sayısı × CHF 215) + (> 12 yaş çocuk sayısı × CHF 268) + (Eğitimdeki kişi sayısı × CHF 268)
3

Tek Seferlik Ödenekler (Doğum/Evlat Edinme)

Zürih Kantonu, bazı Batı İsviçre kantonlarının aksine, tek seferlik doğum veya evlat edinme ödeneği vermez.

Ödenek Türü Tutar (ZH)
Doğum Ödeneği CHF 0
Evlat Edinme Ödeneği CHF 0
4

Yıllık Hakkın Belirlenmesi

Ödeme genellikle işveren aracılığıyla veya doğrudan yetkili Aile Denkleştirme Fonu tarafından aylık olarak yapılır:

Yıllık Tutar = (Aylık Toplam Hak) × 12

3. Teknik Kod Alıntısı (welfarecalc.ch Çekirdek Motoru)

Sunucularımızda ve yerel olarak tarayıcınızda çalıştırılan hesaplama mantığının resmi JavaScript uygulaması aşağıdadır:

// Zürih Kantonu (ZH) Aile Ödenekleri Algoritması 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // 12 yaşına kadar const childOver12PerMonth = 268; // 12 ila 16 yaş const baseEduPerMonth = 268; // 16 yaşından itibaren eğitimde if (numChildren > 0) { // 12 yaşından büyük yaş grubunun savunmacı doğrulaması const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // ZH'de doğum/evlat edinme ödeneği yok return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Hak Sahipliği ve Öncelik:
Aynı çocuk için aile ödeneğini yalnızca bir ebeveyn alabilir. Her iki ebeveyn de çalışıyorsa, FamZG Madde 12'ye göre yasal öncelik sırası, ilk hakkı (çalışma yerine, vesayete veya daha yüksek gelire göre) kesin olarak düzenler.
