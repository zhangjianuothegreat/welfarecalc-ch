En Suiza, las asignaciones familiares están reguladas por ley (LAF). Nuestra calculadora utiliza las disposiciones cantonales específicas del Cantón de Zúrich para 2026 para determinar su derecho con precisión. El Cantón de Zúrich aplica aquí una particularidad con un escalón de edad a partir de los 12 años cumplidos.

1. Tasas mensuales (2026)

Asignación por hijos (hasta 12 años)

CHF 215 / mes

Para hijos desde el nacimiento hasta los 12 años cumplidos.

Asignación por hijos (desde 12 años)

CHF 268 / mes

Escalón de edad específico de Zúrich: Desde los 12 hasta los 16 años cumplidos.

Asignación de formación

CHF 268 / mes

A partir de los 16 años hasta el final de la formación (máximo hasta los 25 años).

2. El método de cálculo

1

Determinación de la categoría y el escalón de edad

El algoritmo verifica la edad exacta y el estado de formación de cada hijo:

  • 0 a 12 años: Derecho a la asignación por hijos regular de CHF 215.
  • 12 a 16 años: Derecho al escalón de edad elevado de Zúrich de CHF 268.
  • 16 a 25 años: Derecho a la asignación de formación de CHF 268 (siempre que se encuentre en formación comprobada).
  • Hijos con incapacidad laboral: Asignación por hijos de CHF 268 hasta los 20 años cumplidos en caso de incapacidad laboral permanente.
2

Cálculo del monto mensual total

La suma se forma linealmente a partir de las respectivas tasas, ya que en el Cantón de Zúrich no existe un aumento para familias numerosas (a partir del 3er hijo):

(Nº hijos ≤ 12 a. × CHF 215) + (Nº hijos > 12 a. × CHF 268) + (Nº en formación × CHF 268)
3

Asignaciones únicas (nacimiento/adopción)

El Cantón de Zúrich, a diferencia de algunos cantones de la Suiza francófona, no concede asignaciones únicas de nacimiento o adopción.

Tipo de asignación Monto (ZH)
Asignación de nacimiento CHF 0
Asignación de adopción CHF 0
4

Determinación del derecho anual

El pago se realiza generalmente de forma mensual a través del empleador o directamente a través de la caja de compensación familiar competente:

Monto anual = (Derecho mensual total) × 12

3. Extracto del código técnico (Motor central de welfarecalc.ch)

Aquí está la implementación oficial en JavaScript de la lógica de cálculo que se ejecuta en nuestros servidores y localmente en su navegador:

// Algoritmo de Asignaciones Familiares del Cantón de Zúrich (ZH) 2026 function calculateZuerichFA(formData, stateData) { const numChildren = parseInt(formData.numChildren || 0, 10); const numInEducation = parseInt(formData.numInEducation || 0, 10); let childMonthly = 0; let eduMonthly = 0; const baseChildPerMonth = 215; // Hasta 12 años const childOver12PerMonth = 268; // 12 a 16 años const baseEduPerMonth = 268; // Desde 16 años en formación if (numChildren > 0) { // Validación defensiva del grupo de edad > 12 años const numOver12 = Math.min( Math.max(0, parseInt(formData.numChildrenOver12 || 0, 10)), numChildren ); const numUpTo12 = numChildren - numOver12; childMonthly = (numUpTo12 * baseChildPerMonth) + (numOver12 * childOver12PerMonth); } if (numInEducation > 0) { eduMonthly = numInEducation * baseEduPerMonth; } const totalMonthly = childMonthly + eduMonthly; const totalYearly = totalMonthly * 12; // No hay asignaciones de nacimiento/adopción en ZH return { totalMonthly: totalMonthly, totalYearly: totalYearly, breakdown: { childMonthly: childMonthly, eduMonthly: eduMonthly, birthAllowance: 0, adoptionAllowance: 0 } }; }
Elegibilidad y prioridad:
Solo uno de los progenitores puede percibir la asignación familiar por el mismo hijo. Si ambos progenitores trabajan, el orden de prioridad legal según el Art. 12 LAF regula estrictamente el derecho primario (según lugar de trabajo, custodia o ingreso mayor).
