2. ¿Qué datos personales tratamos?

Solo tratamos los datos técnicamente necesarios para el funcionamiento del sitio web:

  • Datos técnicos: dirección IP, tipo y versión del navegador, sistema operativo, configuración de idioma, hora de acceso
  • Datos de uso: idioma seleccionado, calculadoras utilizadas (por ejemplo, IPV, EL, Asistencia Social), valores introducidos (ingresos, patrimonio, alquiler, número de hijos, etc.)
Importante: Todos los datos de cálculo que usted introduce (ingresos, patrimonio, primas del seguro médico, datos familiares, etc.) permanecen exclusivamente en su navegador. No almacenamos estos datos en nuestros servidores ni los transmitimos a terceros. El cálculo se realiza íntegramente en su propio dispositivo.

No almacenamos ningún dato personal como nombre, dirección, número AVS, correo electrónico u otra información identificable.

3. Finalidad y base jurídica

El tratamiento se realiza exclusivamente para proporcionar la funcionalidad del sitio web (calculadora de prestaciones sociales) y para mejorar técnicamente el servicio. La base jurídica es el art. 31 de la Ley Federal de Protección de Datos (revDSG) – interés legítimo en el funcionamiento del sitio web.

4. Plazo de conservación

Los archivos de registro del servidor (datos técnicos de acceso) se conservan durante un máximo de 7 a 30 días y luego se eliminan automáticamente. Todos los datos de cálculo permanecen exclusivamente en el navegador del usuario y no se nos transmiten ni se almacenan de forma permanente.

5. Comunicación a terceros y servicios utilizados

Solo compartimos sus datos con terceros en la medida técnicamente necesaria para el funcionamiento del sitio web. No se realiza ninguna comunicación con fines publicitarios o analíticos. No utilizamos ninguna herramienta de seguimiento como Google Analytics, Facebook Pixel o servicios similares.

5.1 Alojamiento

Este sitio web está alojado por Vercel Inc. (EE. UU.), una plataforma global para sitios web estáticos. Vercel opera una red edge mundial con servidores en Europa (incluyendo Fráncfort, Londres), EE. UU. y otras regiones. Cuando visita nuestro sitio, los datos técnicos de acceso (dirección IP, hora de acceso, recursos solicitados) se transmiten a Vercel y se entregan a través de la red edge más cercana.

Vercel está certificado bajo el Marco de Privacidad de Datos UE-EE. UU. y procesa datos personales bajo cláusulas contractuales tipo (CCT) de acuerdo con el RGPD. El código fuente está alojado en GitHub Inc. (EE. UU.), también certificado DPF. Ambos servicios procesan datos únicamente para proporcionar el sitio web y no almacenan datos personales más allá de lo técnicamente necesario.

5.2 Fuentes – Bunny Fonts

Para la visualización de las fuentes, utilizamos Bunny Fonts, operado por bunny.net con servidores en la Unión Europea. Cuando visita el sitio web, su dirección IP se transmite a bunny.net. Bunny Fonts no almacena datos personales con fines de seguimiento y está alojado cumpliendo con las normas de protección de datos dentro de la UE. Para más información: Política de privacidad de bunny.net.

5.3 Generación de PDF – jsPDF a través de Cloudflare CDN

Para la función "Descargar PDF", utilizamos la biblioteca de código abierto jsPDF. La biblioteca se carga a través de la red de distribución de contenido (CDN) de Cloudflare, Inc. (EE. UU.) (cdnjs.cloudflare.com). Durante este proceso, su dirección IP se transmite a Cloudflare. Cloudflare está certificado bajo el Marco de Privacidad de Datos UE-EE. UU. (consulte el Certificado DPF). La biblioteca se ejecuta exclusivamente en su navegador; ninguno de sus datos de entrada (ingresos, patrimonio, asignaciones familiares, etc.) se transmite a Cloudflare ni a ningún otro tercero. El archivo PDF se crea y guarda directamente en su dispositivo.

5.4 Ningún otro servicio de terceros

No integramos ningún otro servicio externo. En particular, no utilizamos Google Fonts, Google Maps, YouTube, complementos de redes sociales ni redes publicitarias.

6. Sus derechos

Según la Ley Federal de Protección de Datos (revDSG) y – cuando corresponda – el Reglamento General de Protección de Datos (RGPD) de la UE, usted tiene los siguientes derechos:

  • Acceso: Puede solicitar información sobre los datos personales que tenemos sobre usted en cualquier momento.
  • Rectificación: Puede solicitar la corrección de datos inexactos.
  • Supresión: Puede solicitar la eliminación de sus datos, siempre que no existan obligaciones legales de conservación que lo impidan.
  • Oposición: Puede oponerse al tratamiento de sus datos por motivos derivados de su situación particular.

Para ejercer sus derechos, contáctenos por correo electrónico a welfarecalc.ch@gmail.com. No hay ningún costo por esto.

7. Seguridad de los datos

Implementamos medidas técnicas y organizativas para proteger sus datos contra accesos no autorizados, pérdida o manipulación. El sitio web se entrega exclusivamente a través de HTTPS (cifrado SSL/TLS). Dado que se trata de un cálculo puramente del lado del cliente, sus datos nunca salen de su navegador.

8. Cambios en esta Política de privacidad

Nos reservamos el derecho de actualizar ocasionalmente esta Política de privacidad para reflejar cambios en los requisitos legales o desarrollos técnicos. La versión actual siempre se puede encontrar en https://www.welfarecalc.ch/privacy.html.
