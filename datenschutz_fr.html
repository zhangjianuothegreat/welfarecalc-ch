2. Quelles données personnelles traitons-nous ?

Nous traitons uniquement les données techniquement nécessaires au fonctionnement du site web :

  • Données techniques : adresse IP, type et version du navigateur, système d'exploitation, paramètres de langue, heure d'accès
  • Données d'utilisation : langue sélectionnée, calculateurs utilisés (par exemple, IPV, EL, Aide sociale), valeurs saisies (revenus, patrimoine, loyer, nombre d'enfants, etc.)
Important : Toutes les données de calcul que vous saisissez (revenus, patrimoine, primes d'assurance maladie, données familiales, etc.) restent exclusivement dans votre navigateur. Nous ne stockons pas ces données sur nos serveurs et ne les transmettons pas à des tiers. Le calcul est effectué entièrement sur votre propre appareil.

Nous ne conservons aucune donnée personnelle telle que nom, adresse, numéro AVS, e-mail ou toute autre information identifiable.

3. Finalité et base juridique

Le traitement est effectué uniquement pour fournir la fonctionnalité du site web (calculateur de prestations sociales) et pour améliorer techniquement le service. La base juridique est l'art. 31 de la loi révisée sur la protection des données (revDSG) – intérêt légitime au fonctionnement du site web.

4. Durée de conservation

Les fichiers journaux du serveur (données techniques d'accès) sont conservés pendant une durée maximale de 7 à 30 jours, puis automatiquement supprimés. Toutes les données de calcul restent exclusivement dans le navigateur de l'utilisateur et ne sont ni transmises ni stockées de manière permanente.

5. Transmission à des tiers et services utilisés

Nous ne partageons vos données avec des tiers que dans la mesure techniquement nécessaire au fonctionnement du site web. Aucune transmission à des fins publicitaires ou analytiques n'est effectuée. Nous n'utilisons aucun outil de suivi tel que Google Analytics, Facebook Pixel ou des services similaires.

5.1 Hébergement

Ce site web est hébergé par Vercel Inc. (États-Unis), une plateforme mondiale pour sites web statiques. Vercel exploite un réseau edge mondial avec des serveurs en Europe (notamment Francfort, Londres), aux États-Unis et dans d'autres régions. Lors de votre visite sur notre site, des données techniques d'accès (adresse IP, heure d'accès, ressources demandées) sont transmises à Vercel et délivrées via le réseau edge le plus proche.

Vercel est certifié dans le cadre du Cadre pour la protection des données UE-États-Unis et traite les données personnelles dans le cadre de clauses contractuelles types (CCT) conformément au RGPD. Le code source est hébergé sur GitHub Inc. (États-Unis), également certifié DPF. Les deux services traitent les données uniquement pour fournir le site web et ne stockent pas de données personnelles au-delà de ce qui est techniquement nécessaire.

5.2 Polices – Bunny Fonts

Pour l'affichage des polices, nous utilisons Bunny Fonts, exploité par bunny.net avec des serveurs dans l'Union européenne. Lorsque vous visitez le site web, votre adresse IP est transmise à bunny.net. Bunny Fonts ne stocke pas de données personnelles à des fins de suivi et est hébergé conformément aux règles de protection des données au sein de l'UE. Pour plus d'informations : Politique de confidentialité de bunny.net.

5.3 Génération de PDF – jsPDF via Cloudflare CDN

Pour la fonction "Télécharger le PDF", nous utilisons la bibliothèque open source jsPDF. La bibliothèque est chargée via le réseau de diffusion de contenu (CDN) de Cloudflare, Inc. (États-Unis) (cdnjs.cloudflare.com). Votre adresse IP est transmise à Cloudflare au cours de ce processus. Cloudflare est certifié dans le cadre du Cadre pour la protection des données UE-États-Unis (voir Certificat DPF). La bibliothèque s'exécute exclusivement dans votre navigateur ; aucune de vos données de saisie (revenus, patrimoine, allocations familiales, etc.) n'est transmise à Cloudflare ou à tout autre tiers. Le fichier PDF est créé et enregistré directement sur votre appareil.

5.4 Aucun autre service tiers

Nous n'intégrons aucun autre service externe. En particulier, nous n'utilisons pas Google Fonts, Google Maps, YouTube, de plugins de réseaux sociaux ou de réseaux publicitaires.

6. Vos droits

Conformément à la loi fédérale sur la protection des données (revDSG) et – le cas échéant – au Règlement général sur la protection des données (RGPD) de l'UE, vous disposez des droits suivants :

  • Accès : Vous pouvez demander à tout moment des informations sur les données personnelles que nous détenons à votre sujet.
  • Rectification : Vous pouvez demander la correction de données inexactes.
  • Effacement : Vous pouvez demander la suppression de vos données, dans la mesure où aucune obligation légale de conservation ne s'y oppose.
  • Opposition : Vous pouvez vous opposer au traitement de vos données pour des raisons tenant à votre situation particulière.

Pour exercer vos droits, contactez-nous par e-mail à welfarecalc.ch@gmail.com. Aucun frais n'est facturé pour cela.

7. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, perte ou manipulation. Le site web est exclusivement diffusé via HTTPS (chiffrement SSL/TLS). Comme il s'agit d'un calcul purement côté client, vos données ne quittent jamais votre navigateur.

8. Modifications de cette politique de confidentialité

Nous nous réservons le droit de mettre à jour occasionnellement cette politique de confidentialité pour refléter les modifications des exigences légales ou les évolutions techniques. La version actuelle est toujours disponible à l'adresse https://www.welfarecalc.ch/privacy.html.
