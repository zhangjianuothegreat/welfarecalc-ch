2. Quali dati personali trattiamo?

Trattiamo esclusivamente i dati tecnicamente necessari per il funzionamento del sito web:

  • Dati tecnici: indirizzo IP, tipo e versione del browser, sistema operativo, impostazioni della lingua, orario di accesso
  • Dati di utilizzo: lingua selezionata, calcolatori utilizzati (ad es., IPV, EL, Assistenza sociale), valori inseriti (reddito, patrimonio, affitto, numero di figli, ecc.)
Importante: Tutti i dati di calcolo inseriti (reddito, patrimonio, premi dell'assicurazione malattia, dati familiari, ecc.) rimangono esclusivamente nel vostro browser. Non memorizziamo questi dati sui nostri server né li trasmettiamo a terzi. Il calcolo avviene interamente sul vostro dispositivo.

Non conserviamo dati personali come nome, indirizzo, numero AVS, e-mail o altre informazioni identificabili.

3. Finalità e base giuridica

Il trattamento viene effettuato esclusivamente per fornire la funzionalità del sito web (calcolatore delle prestazioni sociali) e per migliorare tecnicamente il servizio. La base giuridica è l'art. 31 della legge federale sulla protezione dei dati (nLPD) – interesse legittimo al funzionamento del sito web.

4. Periodo di conservazione

I file di log del server (dati tecnici di accesso) vengono conservati per un massimo di 7–30 giorni, quindi cancellati automaticamente. Tutti i dati di calcolo rimangono esclusivamente nel browser dell'utente e non vengono né trasmessi né conservati in modo permanente.

5. Comunicazione a terzi e servizi utilizzati

Condividiamo i vostri dati con terzi solo nella misura tecnicamente necessaria per il funzionamento del sito web. Nessuna comunicazione a fini pubblicitari o di analisi. Non utilizziamo strumenti di tracciamento come Google Analytics, Facebook Pixel o servizi simili.

5.1 Hosting

Questo sito web è ospitato da Vercel Inc. (USA), una piattaforma globale per siti web statici. Vercel gestisce una rete edge mondiale con server in Europa (tra cui Francoforte, Londra), negli USA e in altre regioni. Quando visitate il nostro sito, i dati tecnici di accesso (indirizzo IP, orario di accesso, risorse richieste) vengono trasmessi a Vercel e consegnati attraverso la rete edge più vicina.

Vercel è certificato nell'ambito del Quadro per la protezione dei dati UE-USA e tratta i dati personali nell'ambito di clausole contrattuali standard (SCC) conformemente al GDPR. Il codice sorgente è ospitato su GitHub Inc. (USA), anch'esso certificato DPF. Entrambi i servizi trattano i dati esclusivamente per fornire il sito web e non conservano dati personali oltre quanto tecnicamente necessario.

5.2 Font – Bunny Fonts

Per la visualizzazione dei caratteri, utilizziamo Bunny Fonts, gestito da bunny.net con server nell'Unione Europea. Quando visitate il sito web, il vostro indirizzo IP viene trasmesso a bunny.net. Bunny Fonts non memorizza dati personali a fini di tracciamento ed è ospitato nel rispetto delle norme sulla protezione dei dati all'interno dell'UE. Per ulteriori informazioni: Informativa sulla privacy di bunny.net.

5.3 Generazione PDF – jsPDF tramite Cloudflare CDN

Per la funzione "Scarica PDF", utilizziamo la libreria open source jsPDF. La libreria viene caricata tramite la rete di distribuzione dei contenuti (CDN) di Cloudflare, Inc. (USA) (cdnjs.cloudflare.com). Il vostro indirizzo IP viene trasmesso a Cloudflare durante questo processo. Cloudflare è certificato nell'ambito del Quadro per la protezione dei dati UE-USA (vedi Certificato DPF). La libreria viene eseguita esclusivamente nel vostro browser; nessuno dei dati inseriti (reddito, patrimonio, assegni familiari, ecc.) viene trasmesso a Cloudflare o a terzi. Il file PDF viene creato e salvato direttamente sul vostro dispositivo.

5.4 Nessun altro servizio di terze parti

Non integriamo altri servizi esterni. In particolare, non utilizziamo Google Fonts, Google Maps, YouTube, plugin di social network o reti pubblicitarie.

6. I vostri diritti

Ai sensi della legge federale sulla protezione dei dati (nLPD) e – se applicabile – del Regolamento generale sulla protezione dei dati (GDPR) dell'UE, avete i seguenti diritti:

  • Accesso: Potete richiedere in qualsiasi momento informazioni sui dati personali che conserviamo su di voi.
  • Rettifica: Potete richiedere la correzione di dati inesatti.
  • Cancellazione: Potete richiedere la cancellazione dei vostri dati, nella misura in cui non vi siano obblighi di conservazione legali contrari.
  • Opposizione: Potete opporvi al trattamento dei vostri dati per motivi derivanti dalla vostra situazione particolare.

Per esercitare i vostri diritti, contattateci via e-mail all'indirizzo welfarecalc.ch@gmail.com. Non sono previsti costi per questo.

7. Sicurezza dei dati

Implementiamo misure tecniche e organizzative per proteggere i vostri dati da accessi non autorizzati, perdita o manipolazione. Il sito web viene distribuito esclusivamente tramite HTTPS (crittografia SSL/TLS). Trattandosi di un calcolo puramente lato client, i vostri dati non lasciano mai il vostro browser.
