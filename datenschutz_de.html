2. Welche Personendaten bearbeiten wir?

Vercel ist nach dem EU-US Data Privacy Framework zertifiziert und verarbeitet personenbezogene Daten im Rahmen von Standardvertragsklauseln (SCCs) gemäss GDPR. Der Quellcode wird auf GitHub Inc.
