2. Que dados pessoais tratamos?

Tratamos exclusivamente os dados tecnicamente necessários para o funcionamento do site:

  • Dados técnicos: endereço IP, tipo e versão do navegador, sistema operativo, configurações de idioma, hora de acesso
  • Dados de utilização: idioma selecionado, calculadoras utilizadas (por exemplo, IPV, EL, Assistência Social), valores inseridos (rendimento, património, renda, número de filhos, etc.)
Importante: Todos os dados de cálculo que introduz (rendimento, património, prémios de seguro de saúde, dados familiares, etc.) permanecem exclusivamente no seu navegador. Não armazenamos estes dados nos nossos servidores nem os transmitimos a terceiros. O cálculo ocorre inteiramente no seu próprio dispositivo.

Não armazenamos quaisquer dados pessoais como nome, morada, número AVS, e-mail ou outras informações identificáveis.

3. Finalidade e base legal

O tratamento é realizado exclusivamente para fornecer a funcionalidade do site (calculadora de prestações sociais) e para melhorar tecnicamente o serviço. A base legal é o art. 31 da lei revista de proteção de dados (revDSG) – interesse legítimo no funcionamento do site.

4. Período de conservação

Os ficheiros de registo do servidor (dados técnicos de acesso) são armazenados por um máximo de 7 a 30 dias e depois automaticamente eliminados. Todos os dados de cálculo permanecem exclusivamente no navegador do utilizador e não são transmitidos nem armazenados permanentemente.

5. Partilha com terceiros e serviços utilizados

Partilhamos os seus dados com terceiros apenas na medida tecnicamente necessária para o funcionamento do site. Não há partilha para fins publicitários ou analíticos. Não utilizamos quaisquer ferramentas de rastreamento como Google Analytics, Facebook Pixel ou serviços similares.

5.1 Alojamento

Este site é hospedado pela Vercel Inc. (EUA), uma plataforma global para sites estáticos. A Vercel opera uma rede edge mundial com servidores na Europa (incluindo Frankfurt, Londres), nos EUA e noutras regiões. Quando visita o nosso site, os dados técnicos de acesso (endereço IP, hora de acesso, recursos solicitados) são transmitidos para a Vercel e entregues através da rede edge mais próxima.

A Vercel está certificada no âmbito do Quadro de Privacidade de Dados UE-EUA e processa dados pessoais no âmbito de cláusulas contratuais padrão (SCC) de acordo com o RGPD. O código-fonte está alojado no GitHub Inc. (EUA), também certificado DPF. Ambos os serviços processam dados exclusivamente para fornecer o site e não armazenam dados pessoais para além do tecnicamente necessário.

5.2 Fontes – Bunny Fonts

Para a exibição das fontes, utilizamos Bunny Fonts, operado por bunny.net com servidores na União Europeia. Quando visita o site, o seu endereço IP é transmitido para bunny.net. Bunny Fonts não armazena dados pessoais para fins de rastreamento e está alojado em conformidade com as normas de proteção de dados dentro da UE. Para mais informações: Política de Privacidade da bunny.net.

5.3 Geração de PDF – jsPDF via Cloudflare CDN

Para a função "Descarregar PDF", utilizamos a biblioteca de código aberto jsPDF. A biblioteca é carregada através da rede de distribuição de conteúdo (CDN) da Cloudflare, Inc. (EUA) (cdnjs.cloudflare.com). O seu endereço IP é transmitido para a Cloudflare durante este processo. A Cloudflare está certificada no âmbito do Quadro de Privacidade de Dados UE-EUA (ver Certificado DPF). A biblioteca é executada exclusivamente no seu navegador; nenhum dos seus dados inseridos (rendimento, património, abonos de família, etc.) é transmitido para a Cloudflare ou para terceiros. O ficheiro PDF é criado e guardado diretamente no seu dispositivo.

5.4 Nenhum outro serviço de terceiros

Não integramos quaisquer outros serviços externos. Em particular, não utilizamos Google Fonts, Google Maps, YouTube, plugins de redes sociais ou redes publicitárias.

6. Os seus direitos

De acordo com a lei suíça de proteção de dados (revDSG) e – quando aplicável – o Regulamento Geral de Proteção de Dados (RGPD) da UE, você tem os seguintes direitos:

  • Acesso: Pode solicitar informações sobre os dados pessoais que temos sobre si a qualquer momento.
  • Retificação: Pode solicitar a correção de dados inexatos.
  • Eliminação: Pode solicitar a eliminação dos seus dados, desde que não existam obrigações legais de conservação em contrário.
  • Oposição: Pode opor-se ao tratamento dos seus dados por motivos relacionados com a sua situação particular.

Para exercer os seus direitos, contacte-nos por e-mail para welfarecalc.ch@gmail.com. Não há qualquer custo associado.

7. Segurança dos dados

Implementamos medidas técnicas e organizacionais para proteger os seus dados contra acessos não autorizados, perda ou manipulação. O site é entregue exclusivamente através de HTTPS (encriptação SSL/TLS). Tratando-se de um cálculo puramente do lado do cliente, os seus dados nunca saem do seu navegador.

8. Alterações a esta Política de Privacidade

Reservamo-nos o direito de atualizar ocasionalmente esta Política de Privacidade para refletir alterações nos requisitos legais ou desenvolvimentos técnicos. A versão atual está sempre disponível em https://www.welfarecalc.ch/privacy.html.
